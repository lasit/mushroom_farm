# Mushroom Farm

Small-scale gourmet mushroom operation in Darwin, Australia.

- **Scale**: 20-50kg/week (starting at 5kg/week)
- **Varieties**: Oyster, Pink Oyster, Lion's Mane
- **Model**: Direct-to-consumer, farm pickup only
- **Philosophy**: Small Giants - intentionally small, excellence over growth
